Data definitions & methodology
Internal reference — PeopleConnect SEO / Martech
Data sources
SnowflakeSales / gross adds data via Supermetrics. Refreshes Monday 6 am. Source: bi_layer_db.prod.exp_daily_sales_metrics
Google Search ConsoleImpressions, clicks, CTR, avg position. Data begins Nov 24, 2025. Source: PC_FIVETRAN_DB.GSC.SITE_REPORT_BY_SITE
BigQuery / GA4Bucket test sessions and purchases. Refreshes daily. Source: ga-bigquery-319917.analytics_472677824.events_*
Metric definitions
Gross addsNet new paid subscriptions attributed to the SEO / Organic channel for the period.
TargetWeekly gross adds goal set by the business. Sourced from Snowflake targets table.
% to goalGross Adds ÷ Target. Values above 100% indicate goal was exceeded.
ImpressionsTotal times a brand URL appeared in Google Search results (GSC).
ClicksTotal clicks from Google Search results to the brand site (GSC).
Avg CTRClicks ÷ Impressions. Calculated from GSC data.
Organic conv rateGross Adds ÷ GSC Clicks. Exception: OmniWatch uses GA4 Organic Sessions as denominator (not GSC clicks) due to channel definition differences.
Avg positionAverage Google Search ranking position for the brand's pages. Lower = better.
Weekly periodFriday–Thursday, anchored to 2026-01-01 (a known Thursday) via modular arithmetic in Snowflake SQL.
OmniWatch special handling
Channel filterOW uses channel__name = 'Organic' (not 'SEO' like other brands). Organic = Direct + Organic Search + Organic Social.
Conv rateOW conversion rate uses GA4 Organic Sessions as denominator, not GSC clicks. Currently a manual input (orange column in Business Outcomes tab).
TotalsOW is excluded from all "Core Brands" totals. It is tracked separately.
GSC data prior to Nov 24, 2025 is unavailable. GA4 data begins Aug 17, 2025. Periods prior to these dates will show null for the respective metrics. The executive summary narrative is generated via the Claude API (Anthropic) using Apps Script connected to this workbook.
